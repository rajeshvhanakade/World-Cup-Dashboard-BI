<h1>World Cup Dashboard BI</h1>

<p>
This project presents an interactive Power BI dashboard that analyzes World Cup player statistics to identify top performers globally. 
The dashboard enables users to explore, compare, and gain insights into player performance using clean and structured data.
</p>

<hr>

<h2>Project Overview</h2>
<p>
The objective of this project is to transform raw World Cup player data into meaningful visual insights. 
Using Python for data cleaning and Power BI for visualization, the project highlights key performance indicators such as goals, assists, matches played, and overall contribution.
</p>

<hr>

<h2>Technologies Used</h2>
<ul>
    <li>Power BI – for dashboard creation and visualization</li>
    <li>Python – for data cleaning and preprocessing</li>
    <li>Pandas – for data manipulation</li>
    <li>CSV Dataset – as the primary data source</li>
</ul>

<hr>

<h2>Features</h2>
<ul>
    <li>Interactive Power BI dashboard</li>
    <li>Player-wise performance comparison</li>
    <li>Top performers identification</li>
    <li>Cleaned and structured dataset</li>
    <li>Visual insights using charts and KPIs</li>
</ul>

<hr>

<h2>Data Processing</h2>
<ol>
    <li>Raw World Cup player data was collected.</li>
    <li>Data was cleaned using Python (handling null values, formatting, and filtering).</li>
    <li>Processed data was exported for visualization.</li>
    <li>Power BI was used to build an interactive dashboard.</li>
</ol>

<hr>

<h2>Dashboard Insights</h2>
<ul>
    <li>Top goal scorers across tournaments</li>
    <li>Player contribution analysis</li>
    <li>Performance trends</li>
    <li>Comparative statistics</li>
</ul>

<hr>

<h2>How to Use</h2>
<ol>
    <li>Download the repository.</li>
    <li>Open the Power BI (.pbix) file.</li>
    <li>Interact with filters and visuals to explore player statistics.</li>
</ol>
